# 比特彗星贴吧解锁版

## 调整内容

相比起官方版 贴吧解锁版调主要整了下列内容：  

* 解锁种子市场  
* 解锁电骡插件  
* 解锁完整IP地址显示  
* 添加tracker服务器并设置更新  
* 删除自动更新程序（其包含捆绑）  
* 调整配置文件以使用系统默认播放器预览视频  
* 调整配置文件以屏蔽迅雷等吸血客户端以及恶意刷流客户端  
* 调整配置文件以平衡下载速度与性能防止内存占用过多或网络中断
